# rfw.sefaz

Biblioteca auxiliar para integrações com os webservices da SEFAZ.

## Codificação de caracteres

Todo o código-fonte é mantido em UTF-8, o que permite o uso transparente de acentos e caracteres específicos do português.

## Desenvolvimento

Este repositório foi originalmente gerado para o ecossistema Eclipse, porém os metadados da IDE não fazem parte do versionamento. Utilize seu ambiente favorito e importe o projeto como um módulo Maven padrão.
